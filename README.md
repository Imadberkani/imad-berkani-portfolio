# Portfolio — Imad Berkani

My personal portfolio website showcasing my projects, skills, and experience as a Data Scientist & ML Engineer.

---

## Built With

- React.js
- React-Bootstrap
- CSS3

---

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Imadberkani/portfolio.git

# Install dependencies
npm install

# Start the development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
